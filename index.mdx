  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-3">Quick start</h2>
  <p className="text-lg text-gray-600 dark:text-gray-400 mb-8">Get up and running with Invoice OCR in minutes</p>
  
  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install the required packages.
      
      ```bash
      git clone https://github.com/bnishit/purchase-ocr.git
      cd purchase-ocr
      npm install
      ```
    </Step>
    
    <Step title="Configure environment variables">
      Copy the example environment file and add your OpenRouter API key.
      
      ```bash
      cp .env.example .env.local
      ```
      
      Add your OpenRouter API key to `.env.local`:
      
      ```bash
      OPENROUTER_API_KEY=your_api_key_here
      OPENROUTER_MODEL=openai/gpt-4o-mini
      ```
    </Step>
    
    <Step title="Start the development server">
      Run the Next.js development server and open your browser.
      
      ```bash
      npm run dev
      ```
      
      Navigate to `http://localhost:3000` and upload your first invoice to extract structured data.
    </Step>
  </Steps>
</div>
